# 基金筛选场景目录

| 场景名 (get_screen_guide 参数) | 描述 | 适用时机 |
|-------------------------------|------|---------|
| concept_exposure | 概念主题曝露度筛选 | 用户提到"概念"、"主题"、"持有XX概念股" |
| industry_exposure | 申万行业曝露度筛选 | 用户提到"行业"、"申万"、"重仓电子/半导体" |
| performance_filter | 跨区间多条件业绩筛选 | 用户提到年化收益/最大回撤/夏普/多区间组合条件 |
| tag_equity | 权益基金标签筛选 | 用户提到"仓位等级"、"择时"、权益基金特征 |
| tag_fixed_income | 固收+基金标签筛选 | 用户提到"固收+"、"稳健/激进"、"转债策略"（仅生产） |
| tag_mixed | 混合基金标签筛选 | 用户提到"混合基金"、"偏股/偏债"（仅生产） |
