# 数据库表目录

| 表名                     | 说明                                                                                                                                                  |
|------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------|
| tb_fd_basic_info       | 基金基础信息：代码、简称、全称、成立/终止日期、三级分类（c_class1/2/3_code/name）、基金经理、托管银行、基金公司、投资范围、申赎状态、基金性质、管理/托管费率                                                          |
| tb_fd_category         | 基金组内分类（自研分类体系）：一级分类（001权益/002固收加/003债券/004混合/005QDII/006FOF/007另类/008货币）、二级分类，按季度截面更新（c_report_date）                                                |
| tb_fd_nav_daily        | 基金每日净值：单位净值(c_nav)、累计净值(c_nav_acc)、复权净值(c_nav_adj)，以及1日/1周/1月/3月/6月/1年/年初至今/去年/年化收益率（小数格式，0.01=1%，展示时×100）                                          |
| tb_fd_asset_allocation | 基金季报资产配置：净值总额(c_fund_nav_total，元)、总资产、股票/债券/现金的市值和占净值比例（单位：%），港股通、可转债子项，报表类型c_style区分季报                                                             |
| tb_fd_portfolio_stk    | 基金季报股票持仓明细：股票代码(c_stk_code)、持仓股数(c_hold_share)、持仓市值(c_hold_value，元)、占净值比例(c_nav_ratio，%)                                                            |
| tb_fd_portfolio_bd     | 基金季报债券持仓明细：债券代码(c_bd_code)、债券名称、债券类型(c_bd_type)、持仓数量(c_hold_num，张)、持仓市值(c_hold_value，元)、占净值比例(c_nav_ratio，%)                                        |
| tb_fd_perform_abs      | 基金绝对收益指标：按区间(c_period_code: 00=近1月/01=近3月/02=近6月/03=近1年/04=近2年/05=近3年/06=近5年/07=年初至今/08=成立以来)的收益率、年化收益、波动率、最大回撤、夏普/卡尔玛/索提诺比率（**已是百分比，25.5表示25.5%**） |
| tb_dict_params         | 通用参数字典：行业分类体系（中信/申万/中证/证监会/GICS等）的代码-名称-层级关系，c_param_type区分不同分类体系，代码层级：6位=一级/9位=二级/12位=三级                                                           |
| tb_fd_tag_asset_eq     | 权益基金资产配置标签：股票仓位均值(c_stk_pos_avg，%)、仓位变动均值、仓位等级（高仓位≥90%/中高仓位<90%）、择时标签（择时/不择时）                                                                       |
| tb_stk_industry        | A股股票行业归属（日频）：股票代码(c_stk_code)、中信行业代码(c_citic_code，三级12位，截断6位=一级/9位=二级)、申万行业代码(c_sw_code)，关联 tb_dict_params 获取行业名称 |
| tb_stk_concept         | A股股票概念归属（日频）：股票代码(c_stk_code)、概念代码(c_concept_code，007前缀)，一股多概念(多对多)，关联 tb_dict_params(c_param_type='概念板块') 获取概念名称，c_remark字段有概念简介 |
